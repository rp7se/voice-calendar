# voice-calendar

面向学习、工作和竞赛场景的语音日程管理系统。

## 项目简介

voice-calendar 是一个以语音交互为核心的日程管理 Web 应用。用户可以通过自然语言快速记录、查看和管理日程，并在日历视图中直观掌握学习、工作与竞赛安排。

当前仓库处于项目初始化阶段，已完成基础工程搭建与首页介绍，具体业务功能将在后续迭代中逐步开发。

## 项目背景

在日常学习、工作和竞赛准备中，日程往往分散在多个工具里，手动录入效率低、查看不够直观。voice-calendar 希望将语音交互与日历管理结合，让用户用更自然的方式管理时间，减少操作成本，提升日程规划的效率与体验。

## 第三方依赖说明

本项目使用以下技术和工具完成开发：

* React：用于构建前端用户界面
* TypeScript：用于增强代码类型约束和可维护性
* Vite：用于项目构建和本地开发环境
* Web Speech API：浏览器原生语音识别与语音播报能力
* localStorage：浏览器本地存储能力，用于保存日程、倒计时和分类数据

本项目未使用第三方日历模板或现成日程管理系统，日历视图、日程数据结构、语音指令解析、倒计时展示、分类管理和日程总结逻辑均围绕本作品需求自行设计和实现。

## 原创功能说明

本项目围绕“语音版日历工具”的主题进行设计，核心原创功能包括：

1. 语音日程管理
   支持用户通过语音添加、查看和删除日程，将语音识别结果转换为具体的日历操作。

2. 自定义日历视图
   实现基础月历展示、日期选中、当天高亮和日程数量提示，为用户提供直观的日程查看方式。

3. 当日详情页
   支持点击某一天查看当天的日程、课程、工作和提醒事项，并统计当天事项数量和预计总耗时。

4. 节日显示
   在日历日期中显示节日信息，提升日历工具的实用性和信息密度。

5. 倒计时卡片
   支持创建考试、比赛、面试等重要事项的倒计时，用更直观的方式提醒用户关键日期。

6. 分类文件夹
   支持将不同类型的日程归类管理，例如学习、比赛、面试、课程和工作，方便用户按场景查看安排。

7. 语音总结与播报
   支持对当天或本周日程进行语音总结，统计每天大致需要投入的时间，并播报当前时间和重点事项。

以上功能的业务逻辑、数据结构、交互流程和页面组织均根据本项目需求设计实现。第三方框架主要用于页面构建和开发环境搭建，不包含直接生成核心业务功能的现成模板。


## 未来核心功能规划

- **语音日程管理**：通过语音添加、查看、删除日程
- **日历与节日**：日历视图展示，支持节日信息显示
- **倒计时卡片**：重要日期倒计时提醒
- **当日详情页**：查看某一天的完整日程安排
- **分类文件夹**：按学习、工作、竞赛等场景分类管理
- **语音总结与播报**：语音总结当日/当周安排，支持语音播报提醒

## 技术栈

- **前端框架**：React 19
- **语言**：TypeScript
- **构建工具**：Vite
- **代码规范**：ESLint
